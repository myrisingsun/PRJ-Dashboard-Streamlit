# Sprint 7 — Результаты

## Страница: Index (Портфель проектов)

### 1. Новый статус «В постоянном режиме»

- Добавлен статус `В постоянном режиме` в `STATUS_COLORS` (цвет `#3498DB`) и `STATUS_EMOJI` (🔵) в `data/loader.py`
- В разделе KPI добавлен 6-й показатель **"Пост. режим 🔵"** (было 5 метрик, стало 6)
- На карточках проектов статус отображается в сокращённой форме: **"В пост. режиме"**

---

### 2. Интерактивные диаграммы распределения

Диаграммы **"Распределение по статусам"** и **"Распределение по приоритетам"** переработаны:

- Тип изменён с pie/donut на **горизонтальные bar-чарты** (pie-чарты не поддерживают `on_select` в Streamlit)
- Добавлена интерактивность через `on_select="rerun"` — клик по бару раскрывает таблицу проектов под диаграммой
- При клике на статус — таблица: Код / Название / Приоритет
- При клике на приоритет — таблица: Код / Название / Статус
- Фиксированный порядок категорий:
  - Статусы (сверху вниз): По плану → В пост. режиме → Есть риски → Отстает → Приостановлен
  - Приоритеты (сверху вниз): Высокий → Средний → Низкий
- Высота диаграмм динамическая: `max(160, n * 44)` px

---

### 3. Финансовые диаграммы

#### Удалено
- Grouped bar-чарт **"Бюджет vs Факт оплат"** (плохо масштабировался из-за INFR.0001)

#### Добавлено: «Исполнение плана оплат, %»

- Горизонтальный bar-чарт: факт / план × 100%
- Ось X фиксирована: 0–110%
- Пунктирная линия на отметке 100%
- Цвет баров: 🟢 зелёный ≤ 100%, 🔴 красный > 100%
- Подписи: `67%  (1.2 млн из 1.8 млн)`
- Сортировка по % исполнения (по возрастанию)
