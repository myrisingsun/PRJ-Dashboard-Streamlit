## 3. Маркеры C (контрольные точки) с popup-комментарием

### Диаграмма Roadmap
- Жёлтый круг `#F39C12` с оранжевой рамкой, `go.Scatter`, `symbol="circle"`, size=14
- Hover-текст: `hovertext` + `hoverinfo="text"` (предформатированный HTML)

### Таблица «Исполнение ключевых работ»
- Символ ◆ в ячейке с `onclick="cpShow(this)"`
- При клике — popup с комментарием из исходника

### Формат ячейки в Google Sheets (лист 03.PRJ_STATUS)

| Значение | Поведение |
|----------|-----------|
| `C` | popup показывает название работы из столбца D |
| `C Текст комментария` | popup показывает «Текст комментария» |

### Реализация popup
- Таблица рендерится через `st.components.v1.html()` (полноценный iframe с JS)
- `<div id="cp-popup">` — фиксированный блок, скрыт по умолчанию
- `cpShow(el)` — позиционирует popup относительно кликнутого элемента через `getBoundingClientRect()`
- Закрытие: кнопка ✕ или клик вне popup
- Высота iframe рассчитывается динамически по числу строк и проектов

## 4. Годовые разделители на диаграмме Roadmap

- **Чередующиеся полосы** (`add_vrect`): лёгкий серый фон для чётных годов
- **Разделительная линия** (`add_vline`): красный пунктир на 1 января каждого нового года
- **Подпись года** (`add_annotation`): жирная цифра над диаграммой, центрирована по видимой части полосы
- `last_year = (x_max − 1 день).year` — исключает пустые годы за пределами данных
- Ось X явно зафиксирована по диапазону данных (`fig.update_xaxes(range=[...])`)
