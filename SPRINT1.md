# Sprint 1 — AFI Project Portfolio Dashboard

**Период:** февраль 2026
**Статус:** ✅ Завершён

---

## Цель спринта

Создать с нуля многостраничный дашборд управления проектным портфелем на базе Streamlit, подключённый к Google Sheets как единому источнику данных, с аутентификацией, визуализацией и аналитикой.

---

## Архитектура

```
Google Sheets (источник данных)
├── 01.PRJ_LIST       — реестр проектов (мастер)
├── 02.OPER_LIST      — операционные задачи вне проектов
├── 03.PRJ_STATUS     — ключевые точки план/факт
├── 04.PRJ_TEAM       — матрица участия команды
└── 05.PRJ_MONEY_2026 — финансы: бюджет, план, факт оплат
        ↓
  gspread 6.x + Google Service Account
        ↓
  @st.cache_data (TTL 5 мин)
        ↓
  data/loader.py — парсинг и агрегация
        ↓
  Pages (5 разделов) — аутентификация на каждой странице
```

**Технологический стек:**
- Python 3.11
- Streamlit ≥ 1.35
- streamlit-authenticator ≥ 0.3.2 (bcrypt, cookie-сессии)
- gspread 6.x (Google Sheets API)
- pandas ≥ 2.0, plotly ≥ 5.18
- Docker + docker-compose (деплой на VPS)

### 📋 Страница 1 — Портфель проектов (Index)

**KPI-строка (5 метрик):**
- Всего проектов
- По плану 🟢 / Есть риски 🔴 / Отстает 🟡 / Приостановлен ⚫

**Финансовая сводка по портфелю** (над карточками):
- Бюджет / План оплат / Факт оплат / Не оплачено
- Прогресс-бар исполнения плана
- Пересчитывается при изменении фильтров

**Карточки проектов:**
- Код проекта + бейдж статуса (цвет по статусу)
- Бейдж приоритета: 🔴 Высокий / 🟡 Средний / ⚫ Низкий
- Название проекта
- Плановый срок + руководитель проекта (из PRJ_TEAM, роль A)
- Описание системы (3 строки, обрезается с ellipsis)
- Финансовый блок: Бюджет / План / Факт с прогресс-баром
- Ссылки: 📋 Bitrix / 📎 Приказ / 🔗 PROD / 🧪 TEST
- Метка «✅ Запущен»
- Равная высота карточек в ряду (CSS flexbox)

**Фильтры (сайдбар):**
- Проекты: мультиселект + кнопка «Все проекты»
- Статус: мультиселект + кнопка «Все статусы»
- Приоритет: мультиселект + кнопка «Все приоритеты»
- Количество карточек в ряду: 1 / 2 / 3

**Сортировка карточек:** Высокий → Средний → Низкий приоритет

**Диаграммы:**
- Круговая — распределение по статусам (donut 40%)
- Круговая — распределение по приоритетам (donut 40%)
- Горизонтальный бар — Бюджет vs План vs Факт по проектам

**Реестр проектов** (таблица без прокрутки):
- Код / Название / Приоритет / Статус / Срок / Описание
- Высота вычисляется динамически по числу строк

## Ключевые технические решения

### Загрузка данных (data/loader.py)
| Функция | Что делает |
|---|---|
| `_load_raw(sheet)` | Сырые данные из Google Sheets, кэш 5 мин |
| `load_prj_list()` | PRJ_LIST, заголовок строка 2, данные с строки 3 |
| `load_prj_team()` | PRJ_TEAM, очистка имён (`_clean_col_name`), дедупликация (`_dedup_names`) |
| `load_prj_status()` | PRJ_STATUS, многоуровневый заголовок (год / квартал / месяц) |
| `load_prj_money()` | PRJ_MONEY, группировка по IT-продукту |
| `get_pm_per_project()` | Извлечение РП (роль A) из PRJ_TEAM → `{code: "ФИО"}` |
| `get_finance_per_project()` | Итого по проекту из строк "Итого помесячно" (col A = код) |
| `_parse_rub(s)` | Парсинг "1 900 000 ₽" → float |
| `parse_date_range(s)` | Парсинг "Q1 2025 – Q4 2026" → (Timestamp, Timestamp) |

**Правило финансов:** берутся только строки, где столбец A = код проекта (формат `XX.0000`) И в столбце B есть «итого». Двойное кэширование убрано — `get_finance_per_project()` запускается при каждой загрузке страницы (только сырые данные кешируются).

| Есть риски | `#E74C3C` (красный) | 🔴 |

### Equal-height карточки (CSS)
Цепочка flexbox от Streamlit-колонки до `.prj-card`:
```
stHorizontalBlock (align-items: stretch)
  → column (flex-direction: column)
    → div (flex: 1)
      → stVerticalBlock (flex: 1)
        → stMarkdownContainer (flex: 1)
          → stMarkdownContainer > * (flex: 1)  ← ключевое звено
            → .prj-card (flex: 1, height: 100%)
              → .prj-desc (flex: 1, min-height: 3.5em)  ← выравнивает пустые карточки
```

## Конфигурация деплоя

### Локальный запуск
```bash
pip install -r requirements.txt
streamlit run app.py --server.port 8501
```
Credentials: `.streamlit/secrets.toml`

### Docker (VPS)
```bash
docker-compose up -d --build
```
Credentials: `.env` файл (не попадает в образ)

### Обновление данных
- Автоматически каждые 5 минут (TTL кэша)
- Вручную: кнопка **🔄 Обновить данные** в сайдбаре (сбрасывает `st.cache_data`)
